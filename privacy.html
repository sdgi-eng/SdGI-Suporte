Política de Privacidade — SdGI

Última atualização: 2026

O aplicativo SdGI (Suporte de Gás Inteligente) foi desenvolvido para monitoramento remoto de equipamentos de medição de peso de recipientes de gás, permitindo visualização de dados, alertas e status do sistema.

1. Dados coletados

O aplicativo pode coletar, quando necessário para funcionamento:

  • Nome do usuário
  • Endereço de e-mail
  • Identificador do usuário
  • Identificador do dispositivo
  • Dados de uso do aplicativo
  • Dados de diagnóstico e falhas
  • Configurações definidas pelo usuário
  • Dados operacionais do equipamento conectado

2. Finalidade do uso

  • Permitir o funcionamento do monitoramento remoto
  • Enviar alertas e notificações
  • Gerenciar a conta do usuário
  • Melhorar desempenho e estabilidade
  • Diagnóstico de erros

3. Serviços de terceiros

O aplicativo utiliza serviços de terceiros para sua operação:

Firebase (Google)
  • Autenticação
  • Banco de dados em nuvem
  • Notificações push
  • Análises e diagnóstico

Esses serviços podem processar dados conforme suas próprias políticas de privacidade.

4. Compartilhamento de dados

Não vendemos nem comercializamos dados pessoais. Os dados não são compartilhados com terceiros para fins de marketing.

5. Armazenamento e segurança

Os dados são armazenados em serviços de nuvem que utilizam medidas de segurança padrão de mercado.

6. Direitos do usuário

O usuário pode solicitar:

  • Acesso aos dados
  • Correção de dados
  • Exclusão de dados

Basta entrar em contato pelo e-mail de suporte.

7. Retenção de dados

Os dados são mantidos apenas pelo tempo necessário para o funcionamento do serviço.

8. Contato

Dúvidas sobre privacidade ou dados pessoais:

Email: suportedegasinteligente@gmail.com

9. Alterações nesta política

Esta política pode ser atualizada periodicamente. A versão mais recente estará sempre disponível nesta página.
